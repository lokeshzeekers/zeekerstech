ZweldMET

AI-Assisted Smart Welding Helmet with Live Vision & Real-Time Guidance

ZweldMET is a custom-designed welding helmet integrated with patented camera and low-latency display technologies that allow welders to view live welding video directly inside the helmet. AI-powered machine vision generates welding paths and guides the welder for seamless operations while displaying critical welding parameters in real time. All events and welding data are securely logged to the cloud with accurate timestamps for remote access and traceability.

Basic Version — Equipped with a patented camera and ultra-low latency display system that enables welders to perform welding operations using live video feed instead of directly viewing through the lens. Critical welding parameters are displayed in real time without alerts.
Prime Version — Includes the patented live vision display system along with real-time monitoring of critical welding parameters, enabling enhanced visibility and operator awareness during welding.
AI-Based Welding Guidance — Machine vision algorithms generate weld paths and assist the welder with real-time directional guidance for accurate and consistent weld quality.
